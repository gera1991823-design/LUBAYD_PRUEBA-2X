GESTIÓN FORESTAL LUBAYD - V13 CHAT MODERN

NOVEDADES
- Chat rediseñado con estilo moderno tipo WhatsApp empresarial.
- Lista de conversaciones con búsqueda, indicador de actividad y filtro de no leídos.
- Burbujas modernas, confirmación visual de envío y separadores por día.
- Diseño adaptable para PC, iPhone y Android.
- Se mantiene toda la lógica de Firebase, usuarios, partes, GPS, gráficos y filtros.

INSTALACIÓN
1. Subir todos los archivos de esta carpeta al repositorio LUBAYD-SA.
2. Reemplazar los archivos anteriores, incluida la carpeta assets.
3. Hacer Commit changes.
4. Esperar la publicación de GitHub Pages.
5. En PC usar Ctrl + F5.
6. En el celular cerrar por completo la PWA y volver a abrirla.
7. Si la versión anterior queda en caché, eliminar el acceso directo y volver a instalarla.

FIREBASE
- No es necesario cambiar la estructura de datos ni las reglas si la V12 ya funcionaba.
- La cuenta principal debe conservar role: admin.
- Los demás usuarios deben conservar role: operador.
